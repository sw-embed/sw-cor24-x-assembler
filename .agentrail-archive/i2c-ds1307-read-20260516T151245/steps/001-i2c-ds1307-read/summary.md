Added src/examples/assembler/i2c_ds1307_read.s — bit-bang i2c demo reading the DS1307 RTC's seconds/minutes/hours registers via the auto-incrementing pointer protocol, formatting BCD → ASCII, and printing 'HH:MM:SS\n' to UART. Halts after one read. ~290 lines (mostly inlined i2c primitives copied verbatim from i2c_add1_ping.s plus new print_bcd_byte helper). Masks CH bit on seconds (0x7F) and 12/24-mode bit on hours (0x3F) per the DS1307 datasheet. Verified end-to-end via cor24-emu --i2c-device ds1307@0x68 producing '00:00:00\n' (default all-zero registers) in 1177 instructions. Registered in tests/integration_tests.rs::examples() between I2C Add1 Ping and Literals. Workspace tests + clippy clean. Discovered the new emulator's no-device-attached SDA-default changed from 1 to 0 (so output is now '00:00:00' rather than 'FF:FF:FF' without --i2c-device); updated the file's header comment accordingly.
